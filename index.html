Repo-backup script generator

If you need to back up all your repos somewhere, and you have a lot of them, cloning them one by one is a pain. And you can forget which repos you've cloned.

This page generates a shell script that clones all your repos in one go. If a repo already exists, it will just pull the latest changes from the default branch.

Note: currently, this only works on Mac and Linux for GitHub repos and up to 100 repos per user or organisation.

Usage

  1. Set yourself up to connect to GitHub with ssh automatically.
  2. To get your organisation's repos, enter its GitHub name below. To get your personal repos, enter your GitHub username.
  3. If you want to include private repos, you'll need a GitHub Personal API token with full repo scope.
  4. Click 'Generate'.
  5. To add another organisation's or user's repos to the same script, add their details to the form and click 'Generate' again.
  6. Copy and paste the text into a .sh or .bat file.
  7. Run that file where you store your repos. They will be cloned there in organisation folders.
